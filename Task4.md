Допустим есть абстрактный класс `Printer` с методом `print`, у которого есть наследник `BlackAndWhitePrinter`. Этот 
класс уже выложен в продакшн, поэтому он закрыт для изменений. Если мы захотим добавить возможность цветной печати, то
вместо модификации класса `BlackAndWhitePrinter`, нужно создать новую реализацию `Printer`, например, `ColorPrinter`.
Таким образом, класс `Printer` открывает возможность для расширения списка реализаций.
