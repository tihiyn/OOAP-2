# Рефлексия

Я плохо знаю Python и могу ошибаться, но в Python есть возможность множественного наследования. А что касается замечания
по поводу того, что можно создать класс, не входящий в нашу систему типов, то так и в Java можно сделать, потому что
наша иерархия искусственная, и мы сами должны контролировать этот момент. Да, есть нюанс, что когда мы создадим один 
единственный инстанс (Void) класса None, то мы можем вызывать любые доступные методы его предков, что, наверное, не 
очень правильно. 

Что касается предложенного решения на Java, то это вообще непонятно что, пыль в глаза. Класс None тут ничего не 
замыкает, потому что нет полноценной иерархии. А если бы она была, то ничего бы не заработало, потому что Java не 
поддерживает множественное наследование.
