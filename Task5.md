В Java поддерживаются все 5 принципов повторного использования кода:
1. Класс, параметризованный типом/типами. 
2. Класс, в котором методы обращаются друг к другу.
3. Например, класс "Улучшатель фото" с методом "улучшить". Одна реализация улучшает свет, другая улучшает черты лица, 
другая улучшает фон.
4. Классический пример полиморфизма, где вместо `List` можно использовать любую его реализацию.
5. Частично реализованные абстрактные классы.
