1. Для использования класса, как модуля, в другом классе, в Java есть ключевое слово `import`.
2. В Java есть пакеты, которые служат для группировки классов. Есть возможность настраивать область видимости классов, 
относительно пакетов. Есть возможность импортировать пакеты целиком.
3. В Java есть отдельная структурная единица, которая так и называется - `module`. Система модулей помогает группировать
пакеты по их смыслу и задачам. Главное преимущество такого подхода в том, что для каждого модуля можно задавать то, 
какие зависимости необходимы модулю, и то, какие пакеты он "выставляет наружу", то есть делает доступными для других 
модулей.
4. В экосистеме Java существуют различные системы сборки (Maven, Gradle), которые позволяют использовать в своей 
программе другие программы (внешние библиотеки).
